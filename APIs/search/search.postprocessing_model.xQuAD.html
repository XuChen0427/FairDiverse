calculate_xquad_score(qid, bm25_dict, _lambda=0.5)[source]¶

Implements the xQuAD algorithm for diversified document ranking.

Parameters:

  • qid – Query ID for which documents are being ranked.

  • bm25_dict – Dictionary containing BM25 scores for both original and suggestion queries.

  • _lambda – A trade-off parameter (0 to 1) that balances relevance and diversity. A higher value favors relevance, while a lower value emphasizes diversity.

Returns:

A list of selected document IDs in diversified ranking order.

rerank(config)[source]¶

Re-ranks documents using the xQuAD algorithm based on BM25 scores and query diversification.

Parameters:

config – A dictionary containing configuration parameters.
